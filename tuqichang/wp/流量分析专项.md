## 无线流量  --ctf

1、使用airdecap爆破得到密码

2、使用密码生成新文件得到flag

打开流量包如下所示：

![image-20260406191934431](../image/image-20260406191934431.png) 

说明为wifi无线流量

使用aircrack破解密码

```shell
user@WIN-5AM5FNE9G4E:~$ aircrack-ng -w "/mnt/d/CTF/爆破字典/弱口令字典/passwd-CN-Top10000.txt" "/mnt/d/CTF-L/流量分析/1.以前及收集/1-20/11.无线流量/11.无线流量/ctf.pcap"

格式：aircrack-ng -w "字典" "流量包"
```

破解后得到密码

<img src="../image/image-20260406192208906.png" alt="image-20260406192208906" style="zoom: 50%;" /> 

```shell
wifi名称：ctf
密码：password1
```

再次使用aircrack破解出流量包

```shell
user@WIN-5AM5FNE9G4E:~$ airdecap-ng -e "ctf" -p password1 "/mnt/d/CTF-L/流量分析/1.以前及收集/1-20/11.无线流量/11.无线流量/ctf.pcap"

格式： airdecap-ng -e "WiFi名称" -p WiFi密码 "流量包“
```

解出来后生成一个新的流量包

![image-20260406192506179](../image/image-20260406192506179.png) 

搜索后得到flag

```shell
flag{H4lf_1s_3n0ugh}
```
